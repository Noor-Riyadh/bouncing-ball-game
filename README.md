# 🎮 Bouncing Ball Game

A browser-based interactive game built with **HTML5 Canvas** and **vanilla JavaScript** — no frameworks, no dependencies.

---

## 📌 Features

### Part 1 — Canvas Animation
- Blue ball moves continuously inside the canvas
- Automatically bounces off all 4 walls (top, bottom, left, right)
- Glowing trail effect follows the ball

### Part 2 — User Controls
- **Arrow buttons** (on screen) to nudge the ball's direction
- **Keyboard arrows** also supported
- **Pause / Resume** button to stop and continue the animation
- **Spacebar** also toggles pause

### Part 3 — Speed Control
- Slider to adjust speed from **1x to 5x**
- Speed updates in real time

### Part 4 — Obstacles
- 5 obstacles inside the canvas (rectangles + circles)
- Ball bounces off obstacles on collision
- Player **loses a life** on each obstacle hit
- Game ends at **0 lives** with a Game Over screen

## 🚀 How to Run

No installation needed. Just open the file in any browser:

```bash
# Option 1 — double click the file
bouncing_ball_game.html

# Option 2 — open with VS Code Live Server
Right click → Open with Live Server
```

Or play it live on GitHub Pages:
